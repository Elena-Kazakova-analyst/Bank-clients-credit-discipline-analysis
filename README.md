# Анализ кредитных клиентов: Сегментация и портреты заемщиков

Анализ данных клиентов банка для выделения портретов заемщиков в зависимости от их кредитной дисциплины.

## 📋 Описание проекта
Проведен полный цикл анализа данных клиентов банка (50 224 записи, 14 признаков) для выявления ключевых характеристик и различий между группами заемщиков с разной дисциплиной выплат по кредиту (`BAD`, `GOOD`, `MIDDLE`).

**Цель:** Сегментировать клиентов, выявить факторы, коррелирующие с просрочками, и составить детальные портреты каждой группы для минимизации кредитных рисков и улучшения стратегий привлечения.

## 🛠 Стек технологий
- **Python:** `pandas`, `numpy` (обработка данных), `matplotlib`, `seaborn` (визуализация), `datetime` (работа с датами)
- **Jupyter Notebook:** Интерактивная среда разработки
- **Git/GitHub:** Контроль версий и хостинг проекта

## 🚀 Ключевые этапы работы

### 1️⃣ Предобработка данных (Pandas)
- Удаление дублирующихся и производных столбцов (`средняя_сумма_перевода`, `полная_сумма_перевода`)
- Корректировка формата столбца `дата_рождения` к типу `datetime`
- Исправление некорректных значений в признаке `оператор_связи`
- Обработка аномалий: выявление выбросов в числовых признаках с помощью `boxplot` и их фильтрация на основе перцентилей (потеряно **~2.53%** данных, что является допустимым)
- Аномальные наблюдения выделены в отдельный набор для возможного дополнительного исследования

### 2️⃣ Генерация новых признаков (Feature Engineering)
- **Возраст клиента:** Рассчитан на основе даты рождения относительно 2025 года

### 3️⃣ Исследовательский анализ и сегментация (EDA)
- Разделение клиентов на три сегмента по целевому признаку: `BAD` (проблемные), `GOOD` (надежные), `MIDDLE` (средняя дисциплина)
- Сравнительный анализ числовых признаков (средние/медианные значения) в разрезе трех групп
- Сравнительный анализ категориальных признаков (расчет процентных долей) в разрезе групп

### 4️⃣ Визуализация и интерпретация (Matplotlib/Seaborn)
- Гистограммы распределения числовых признаков для сравнения всех трех групп
- Столбчатые диаграммы для категориальных признаков, показывающие долю каждой группы клиентов

### 💡 Ключевые инсайты
- **Сильные предикторы:** Основными факторами, отличающими надежных клиентов, являются **высокая транзакционная активность** (больше переводов на крупные суммы) и **более старший возраст**.
- **Качество трафика:** Клиенты, привлеченные через **«Офис»**, демонстрируют наилучшую платежную дисциплину. Каналы **«Партнер»** и **«Стойка»** дают больший процент проблемных заемщиков.
- **Гендерный фактор:** Пол клиента **не влияет** на его кредитную дисциплину — распределение мужчин и женщин по группам `BAD`, `GOOD`, `MIDDLE` практически идентично.

### 📌 Рекомендации для бизнеса
- **Для кредитного конвейера:** Учитывать возраст и транзакционную активность как дополнительные веса в скоринговых моделях.
- **Для отдела маркетинга:**
  - **Сегмент GOOD:** Предлагать продукты с повышенным кредитным лимитом, премиальные карты и бонусные программы лояльности.
  - **Сегмент MIDDLE:** Разработать предложения для "апгрейда" до категории GOOD (например, снижение ставки при увеличении оборотов).
  - **Сегмент BAD:** Рассмотреть возможность предложения кредитных продуктов с более низкими лимитами и повышенным контролем либо отказ в выдаче крупных сумм.
- **Для отдела привлечения:** Пересмотреть эффективность каналов «Партнер» и «Стойка». Возможно, клиенты из этих каналов требуют более тщательной проверки на входе или для них нужны отдельные, более консервативные кредитные предложения.
